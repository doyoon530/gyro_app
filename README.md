### 기능 설명<br>
**연결 확인** : 워치와 페어링 상태를 확인하는 버튼입니다.<br>
**저장 경로 설정** : xlsx파일 저장 경로를 설정해주는 버튼입니다.<br>
**레이블 설정** : 저장될 레이블을 설정하는 버튼입니다.<br>
**기록 시작** : 엑셀파일 기록을 시작하는 버튼입니다.<br>
**기록 중지 및 저장** : xlsx파일 기록을 중지하고 저장을 하는 버튼입니다.<br>
